Rhizomik CopyrightLY

Decentralized Application (ÐApp) for Copyright Management

Smart contracts are deployed on Rinkeby testnet. A browser providing an Ethereum wallet, or an iOS or Android wallet application, is required. You can try MetaMask.

Features

CopyrightLY is a decentralised application that leverages blockchain and semantic web technologies to facilitate copyright management:

  • Claim content authorship and provide supporting evidence, including existing social media like YouTube videos.
  • Token curated copyright claims based on CLY token staking, which is minted using a bonding curve.
  • License content using NFTs, with semantic metadata stating the reuse terms being transferred.
  • Trade NFTs on existing marketplaces (OpenSea, Rarible...), purchasers can trace copyright to authorship claims.

Scenario & Demo

  1. A YouTube video creator willing to explore alternative ways to monetize content.
  2. Uploads video file and registers content hash on-chain making an authorship claim.
  3. Links authorship claim to YouTube video proofing ownership of the social media platform version.
  4. Stakes CLY token on authorship claim to show commitment on its authenticity.
  5. Mints NFT licensing the reuse terms defined in its metadata to future owners.
  6. Trades NFT on existing marketplace, where potential purchasers can trace it back to the copyright claim.

Implementation

Creators upload their content to decentralised storage and use the resulting content hash to register on-chain authorship claims, which tie content to their identities and a timestamp.

Creators also provide evidence to support their claim, by uploading content or linking videos to an existing version they have published on YouTube. They proof ownership of the social media platform version of the video using an oracle.

Creators and other stakeholders curate the authorship claims by staking the CLY token on them. CLY is minted using a bonding curve. CLY cannot be held, it is minted to be staked on an authorship claim and burned when un-staked.

Claims and evidence can be used in case of a copyright complaint, even in court. This approach also discourages false claims, which cannot be retracted once on-chain.

Content can be monetized using NFTs that state the reuse terms being transferred with the token. Terms are modelled using semantic technologies and based on the Copyright Ontology, which provides the necessary building blocks to make these terms machine-actionable.
